# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | ✅ Active development |

## Reporting a Vulnerability

Если вы нашли уязвимость, пожалуйста, **не создавайте public issue**.

Вместо этого свяжитесь с maintainer'ами через GitHub Security Advisory или
напишите на email, указанный в профиле репозитория.

Мы постараемся ответить в течение 48 часов.
